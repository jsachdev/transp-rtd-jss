TRANSP help documentation
=========================

**TRANSP**
This document contains information on how to maintain and operate the time dependent tokamak transport and data analysis code known as TRANSP.

Additional information can be found at `TRANSPhub <https://www.github.com/PrincetonUniversity/TRANSPhub>`_, a github repository for tracking TRANSP related issues. It is a private repository, send an email to transp_support@pppl.gov to request access. 

Other useful TRANSP related websites include the main `TRANSP website <https://transp.pppl.gov>`_, the old `TRANSP website <https://transpweb.pppl.gov>`_, and the original documentation aka `TRANSP help <https://w3.pppl.gov/~pshare/help/transp.htm>`_.

.. note::

   This project (and this documentation in particular) is under active development.
