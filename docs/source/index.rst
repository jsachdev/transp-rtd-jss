TRANSP help documentation
=========================

**TRANSP**
This document contains information on how to maintain and operate the time dependent tokamak transport and data analysis code known as TRANSP.

Additional information can be found at TRANSPhub, a github repository for tracking TRANSP related issues. It is a private repository, send an email to transp_support@pppl.gov to request access. 

`TRANSPhub <https://www.github.com/PrincetonUniversity/TRANSPhub>`

Other useful links:
- `TRANSP website <https://transp.pppl.gov>`
- `Old TRANSP website <https://transpweb.pppl.gov>`
- `TRANSP help <https://w3.pppl.gov/~pshare/help/transp.htm>`


Check out the :doc:`usage` section for further information, including how to :ref:`installation` the project.

.. note::

   This project is under active development.
